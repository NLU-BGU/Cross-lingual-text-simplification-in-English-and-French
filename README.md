# Cross-lingual Text Simplification in English and French

This repository contains code for the paper **"Translate or Simplify First: An Analysis of Cross-lingual Text Simplification in English and French"**. The project explores different strategies for simplifying text across languages using various prompting techniques and neural models.

## Data

[Access the datasets here](https://drive.google.com/drive/folders/1fdiRgkEusg3f5fGoYZVgP3TwbNpL3nZw?usp=sharing)

## Project Structure

```
├── prompting_strategies.py      # LLM-based simplification using different prompt strategies
├── mT5.py                       # mT5 model training
├── texts_translations.py        # Cross-lingual text translation
├── linguistic_features.py       # Linguistic features analysis
├── automatic_metrics.py         # Evaluation metrics computation
├── requirements.txt             # Package requirements file
└── README.md                    # This file
```
